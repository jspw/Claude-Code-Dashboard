# Code of Conduct

## Our commitment

Claude Code Dashboard is an open, welcoming project. Everyone participating — through issues, pull requests, discussions, or any other channel — is expected to treat others with respect.

## Expected behavior

- Be kind and constructive in all communication
- Give and accept feedback graciously
- Focus on the work, not the person
- Welcome contributors of all experience levels

## Unacceptable behavior

- Personal attacks, insults, or dismissive comments
- Harassment of any kind, public or private
- Sustained disruption of discussions

## Enforcement

Maintainers may remove comments, close issues, or ban users who repeatedly violate these standards. Reports can be made by opening a GitHub issue or contacting the maintainer directly via the repository profile.

---

This project is small and community-driven. Good faith and common sense go a long way.
